Artificial Intelligence for the Semantic Web

The way we search for information is inefficient.

Currently, we rely on ourselves to distill the information we desire into a set of keywords that hopefully returns to us a series of links that we have to spend time browsing in order to determine if we've discovered what we were originally looking for. We at Ada think searching for information should be more like a conversation. We are working on an AI agent who can do the browsing/filtering for us.

Ada is a work-in-progress.

We are currently building the machine learning engine behind Ada:
  • AdaNLP: Natural Language Processing Engine
  • AdaGraph: Graphical Modeling Engine
