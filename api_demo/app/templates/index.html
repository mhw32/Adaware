AdaNLP

A suite of deep learning NLP tools

About

AdaNLP provides a set of natural language processing tools built on neural network models. For example, AdaNLP has tooling to derive base forms of words, parts of speech, name-entity recognition, co-references (coming soon!), and semantic dependencies within text.

AdaNLP was inspired by Stanford CoreNLP and created as a learning exercise as well as an attempt to use only deep learning architectures to compete with the graphical models commonly used in NLP. Each part of AdaNLP was tested on the Brown/WallStreet dataset and achieved state-of-the-art AUC and ACC scores.
